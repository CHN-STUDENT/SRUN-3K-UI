# SRUN-3K-UI
SRUN-3K登陆器UI界面学习

校园网登录器 Qt (C++) 版本

美化的界面，自定义服务器信息，保存账户文件，查看公告等。


使用渣渣的 C++ 知识 + Qt GUI 制作

本程序受大佬学长们的 Python , Html , Java , C#版启发制作。
